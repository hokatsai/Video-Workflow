# 工具：影片壓縮工具 (video_compressor)

## 用途

此工具透過重新編碼並降低影片的位元率（bitrate），來顯著減小影片檔案的體積。這對於在進行分析前預處理大型影片檔案，以提升後續流程的效率非常有幫助。

## 使用方式

您需要提供來源影片檔案、目標輸出路徑，並可選擇性地指定壓縮後的位元率。

### 指令範例

```bash
# 使用預設的 500k 位元率壓縮影片
python 01-system/tools/ops/video_compressor.py "原始影片.mp4" "壓縮後的影片.mp4"

# 指定 1000k 的位元率進行壓縮
python 01-system/tools/ops/video_compressor.py "原始影片.mp4" "壓縮後的影片.mp4" --bitrate "1000k"
```

### 參數說明

*   `input_path` (必須): 來源影片檔案的完整路徑。
*   `output_path` (必須): 您希望保存壓縮後新影片檔案的完整路徑。
*   `--bitrate` (可選): 目標影片位元率，例如 `"500k"`, `"1000k"`, `"2000k"`。位元率越低，檔案越小，但畫質也越差。如果未指定，預設為 `"500k"`，這是一個適用於快速分析的低品質設定。
