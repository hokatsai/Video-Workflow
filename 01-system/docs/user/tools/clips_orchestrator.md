# 工具：影片片段總调度器 (clips_orchestrator)

## 用途

此工具自動化從單一來源影片中提取、調整（適配）和導出多個影片片段的過程。它將依序呼叫 `video_cutter` (影片剪輯工具)、`clips_adapter` (影片片段適配器)，並最終實現類似 `clips_copy_exporter` (片段複製與導出) 的導出功能。

## 使用方式

您可以透過提供原始影片、包含時間戳的 JSON 檔案、基礎輸出目錄，以及可選的解析度和影格率來啟動此總调度流程。

### 指令範例

```bash
python 01-system/tools/ops/clips_orchestrator.py \
    "輸入影片.mp4" \
    "時間戳.json" \
    "輸出目錄/" \
    --resolution "1280x720" \
    --fps 30
```

### 參數說明

*   `input_video` (必須): 來源影片檔案的完整路徑。
*   `timestamps_file` (必須): 包含影片片段時間戳的 JSON 檔案路徑。該 JSON 檔案的格式應為一個物件列表，每個物件代表一個片段：
    ```json
    [
        {"start": 0.0, "end": 10.0, "name": "片頭"},
        {"start": 15.0, "end": 30.0, "name": "主要場景"},
        {"start": 35.5, "end": 40.0, "name": "片尾"}
    ]
    ```
    *   `start`: 片段的開始時間（秒，可為浮點數）。
    *   `end`: 片段的結束時間（秒，可為浮點數）。
    *   `name`: 片段的名稱（字串），將用於命名輸出檔案。
*   `output_base_dir` (必須): 所有處理後的影片片段將被保存到的基礎輸出目錄。
*   `--resolution` (可選): 目標解析度，格式為 `"寬度x高度"` (例如 `"1920x1080"`)。若指定，所有剪輯出的影片片段將被調整到此解析度。
*   `--fps` (可選): 目標影格率，為一個整數 (例如 `24`, `30`)。若指定，所有剪輯出的影片片段將被調整到此影格率。
