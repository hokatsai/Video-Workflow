# 工具參考區

此資料夾專門存放供代理閱讀的工具說明書，內容聚焦於：

- 執行條件：需要的 CLI、腳本、輸入與輸出路徑。
- 操作流程：標準步驟、常見旗標與範例命令。
- 記錄要求：與 `03-outputs/<tool>/...`、`SYSTEM_MEMORY.md` 的掛勾方式。

維護守則：

1. 每個工具建立一份 `*.md` 說明；名稱以工具 ID 為主（例如 `gemini-cli.md`）。
2. 改動後立即更新 `INDEX.md` 以方便索引。
3. 如需新格式，複製 `TEMPLATE.md` 調整；避免混用語言，統一使用繁體中文。
4. 若工具同時提供 user-facing 文件，請在此引用並補充進階技巧。
